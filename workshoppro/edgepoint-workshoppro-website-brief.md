# EdgePoint WorkshopPro — Marketing Website Build Brief

## Project Overview
Build a **modern, premium SaaS marketing website** for **EdgePoint WorkshopPro**, a cloud-based automotive workshop management platform built by **EdgePoint**.

This website is for **automotive mechanics and workshop owners** in **Auckland, New Zealand**, and should be designed to:

- generate demo enquiries
- present the platform as a serious SaaS product
- rank well for NZ automotive workshop software searches
- be easy for search engines and AI answer engines to understand
- feel more modern than competitors like MechanicDesk

The site should present **one single SaaS platform offering**.
There are **no multiple plans or pricing tiers** on the website.
The main goal is to push visitors toward **Book a Demo**.
